# OpenAMP

#### 介绍
本代码仓库主要包含OpenAMP Project相关软件包，包括：

1. openamp  跨运行时/OS通信框架

本代码仓库中软件包主要面向嵌入式场景，主要配合openEuler Embedded使用。和openEuler Embedded
整合通过基于Yocto的[meta-embedded](https://gitee.com/vonhust/yocto-meta-embedded)完成
